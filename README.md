Bluetooth NFC Scanner Keyboard Emulator

This device reads NFC data then prints it like a bluetooth keyboard then virtually presse "Enter" after each read + send.

Thios device also charges 3.7 LiPo Batteries when the power switch is ON and the cable is plugged in
